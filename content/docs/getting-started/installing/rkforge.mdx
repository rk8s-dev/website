## Installing Rkforge

rkforge is a Docker-compatible container image builder and OCI registry management tool. See the [rkforge documentation](/docs/rkforge) for detailed usage information.

### Build from Source

```bash
cargo build -p rkforge --release
```

The binary will be at `./target/release/rkforge`.

### Pre-built Binary

Download the pre-built binary from the [GitHub Releases](https://github.com/rk8s-dev/rk8s/releases) page.
